# SA — 报价单流程规范

> ✅ 初始版本（2026-06-11） | 待用户逐一确认

---

## 一、报价单状态流转

```
草稿 ──▶ 待审核 ──▶ 审核中 ──▶ 已通过 ──▶ 已发送客户 ──▶ 客户已确认
  │                    │         │
  │                    │         └──▶ 已驳回 → 草稿（可重新提交）
  │                    │
  │                    └──▶ 已取消（终态）
  │
  └──▶ 已过期（终态，超过有效期未发送）
```

---

## 二、报价单状态枚举

| 状态 | key | 说明 |
|------|-----|------|
| 草稿 | `draft` | 初始状态，可编辑 |
| 待审核 | `pending` | 已提交，等待审核 |
| 审核中 | `reviewing` | 审核进行中 |
| 已通过 | `approved` | 审核通过，可发送客户 |
| 已驳回 | `rejected` | 审核不通过，可修改后重新提交 |
| 已发送客户 | `sent` | 已发送给客户，等待客户确认 |
| 客户已确认 | `confirmed` | 客户确认报价 |
| 已过期 | `expired` | 超过报价有效期未发送 |
| 已取消 | `cancelled` | 主动取消 |
